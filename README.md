# Customer Behaviour Analysis
##  Project Overview
This is an end-to-end data analytics project focused on analyzing customer purchasing behaviour to generate meaningful business insights. The project follows a structured workflow including data cleaning, analysis, and visualization.

##  Data Cleaning (Python - Pandas)
The dataset was cleaned and preprocessed using Python (Pandas).
* Handled missing values and duplicate records
* Standardized column names and data formats
* Removed inconsistencies to ensure data quality

## Data Analysis (SQL)
SQL was used to perform data analysis and extract insights from the dataset.
* Performed data exploration using SELECT queries
* Used aggregate functions (SUM, AVG, COUNT) for analysis
* Applied GROUP BY and filtering to analyze trends
* Used CASE statements for customer segmentation
* Analyzed product performance, discount impact, and revenue distribution

## Data Visualization (Power BI)
Interactive dashboards were created in Power BI to visualize the insights.
* Customer Analysis Dashboard (demographics, revenue distribution)
* Sales Insights Dashboard (top/bottom products and locations)
* Used charts, KPIs, and filters for better understanding
* 
##  Key Insights
* High-frequency customers contribute significantly to revenue
* Discounts influence purchasing behaviour
* Certain product categories generate higher sales
* Revenue varies across locations and customer segments
* 
## Conclusion
This project demonstrates how data analytics tools like Python, SQL, and Power BI can be used together to transform raw data into actionable insights and support data-driven decision-making.
