# customer_behaviour_analysis
The Customer Behaviour Analysis project is an end-to-end data analytics project that aims to understand customer purchasing patterns and business performance.

The project begins with data extraction and analysis using SQL, followed by data cleaning and preprocessing in Python (Pandas) (Jupyter Notebook). Finally, interactive dashboards are created in Power BI to visualize key insights.

The dashboards provide a comprehensive view of:

* Customer demographics (age, gender)
* Revenue distribution across categories and locations
* Product performance (top and bottom items)
* Impact of discounts and purchasing trends

This project helps in identifying high-performing products, understanding customer segments, and supporting data-driven decision-making for business growth.
